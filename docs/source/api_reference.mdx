## get_endpoint_metrics

<a id="opIdget_endpoint_metrics"></a>

> Code samples

```python
import requests
headers = {
  'Accept': 'application/json',
  'Authorization': 'Bearer {access-token}'
}

r = requests.get('/endpoint/{name}/metrics', params={
  'query': undefined,  'start': undefined,  'end': undefined,  'step': undefined
}, headers = headers)

print(r.json())

```

`GET /endpoint/{name}/metrics`

*Endpoint aggregated metrics*

Endpoint aggregated metrics

Query params are the same as for the `query_range` endpoint, cf <a href='https://prometheus.io/docs/prometheus/latest/querying/api/#range-queries'>Range Queries</a>

<h3 id="get_endpoint_metrics-parameters">Parameters</h3>

|Name|In|Type|Required|Description|
|---|---|---|---|---|
|name|path|string|true|Endpoint name|
|query|query|undefined|true|`query=<string>`: Prometheus expression query string|
|start|query|undefined|true|`start=<rfc3339 | unix_timestamp>`: Start timestamp, inclusive.|
|end|query|undefined|true|`end=<rfc3339 | unix_timestamp>`: End timestamp, inclusive.|
|step|query|undefined|true|`step=<duration | float>`: Query resolution step width in duration format or float number of seconds.|
|timeout|query|string|false|`timeout=<duration>`: Evaluation timeout. Optional. Defaults to and is capped by the value of the `-query.timeout` flag.|

> Example responses

> 401 Response

```json
{
  "error": "string"
}
```

<h3 id="get_endpoint_metrics-responses">Responses</h3>

|Status|Meaning|Description|Schema|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|Endpoint aggregated metrics returned successfully|None|
|401|[Unauthorized](https://tools.ietf.org/html/rfc7235#section-3.1)|A valid user or organization token, sent as a Bearer, is required|[ApiResponseError](#schemaapiresponseerror)|
|404|[Not Found](https://tools.ietf.org/html/rfc7231#section-6.5.4)|Endpoint not found|[ApiResponseError](#schemaapiresponseerror)|
