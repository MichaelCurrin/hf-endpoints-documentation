# How pricing works

With Inference Endpoints, you can easily deploy machine learning models on dedicated hardware. When you create an Endpoint in Inference Endpoint, you configure the instance type, which will be used to power your models. You are billed for the computing and memory resources used when your endpoint is up and running. This means as long as your Endpoint is in the “running” state, you will pay for computing capacity by the hour.

## Pricing Example

| Example 1: Simple Endpoint                                                                                                                                                                                                                                                                                                         |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Endpoint configuration:**<br>-Cloud: AWS<br>Region: us-east-1<br>Instance type: CPU - medium (0.12\$/h)<br>Autoscaling:<br>Min replica: 1<br>Max replica: 1<br>Calculation<br>Instance type * (hours * # min replica) <br>Hourly cost:<br>0.12\$/h _ (1h _ 1) = 0.13$ / h<br>Monthly cost:<br>0.12\$/h * (730h * 1) = 87.6$ / month |

| <br>Example 2: Advanced Endpoint with Autoscaling                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoint configuration:<br>Cloud: AWS<br>Region: us-east-1<br>Instance type: GPU - small (0.6\$/h)<br>Autoscaling:<br>Min replica: 1<br>Max replica: 3<br>Each hour there is a spike of load where the endpoint scales to 3 replica for 15 minutes<br>Calculation<br>Instance type * ((hours * # min replica) + (scale-out hours * # addional replica) ) <br>Hourly cost:<br>0.6\$/h * ((1h * 1) * (0,25 * 2)) = 0.9$ / hour<br>Monthly cost:<br>0.6\$/h * ((730h * 1) * (182,5h * 2))  = 657$ / month |
