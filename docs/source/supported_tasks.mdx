# Supported Transformers Tasks

Below is a table of Hugging Face managed supported tasks for Inference Endpoint. These tasks don't require any form of code or [custom container]([“Create your own inference handler”](/docs/guides/docs/guides/custom_container)) to deploy an Endpoint. 
If you want to customize any of the tasks below, or want to write your own custom task, check out the [“Create your own inference handler”](/docs/guides/docs/guides/custom_handler) section for more information.
